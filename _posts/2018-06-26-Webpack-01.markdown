# npm 설명
## install --save와 --save-dev
<p>
--save는 앱 구동시에 필요한 것들이다. ex) react, vue <br/>
--save-dev는 개발시에 필요한 것들이다. ex) live reload, dev tools, testing tools <br/>
--save를 하면 package.json에 디펜던시가 추가된다.
</p>
